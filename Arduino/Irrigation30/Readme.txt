Прошивка для esp8266
Под ардуинополивалку 3.0

Файловая система LittleFS
Для загрузки файлов используется Uploader LittleFS
Настройки пишутся в EEPROM

Использование:
1. Загрузить файлы из папки data (ctrl+shift+P)
2. Загрузить прошивку
3. Прижать к земле пин GPIO14
4. Перезагрузить контроллер
Должна появиться wi-fi точка доступа AccessPointAuto

С компьютера или телефона подключиться к wifi-сети AccessPointAuto.
web-интерфейс находится по адресу 192.168.4.1

Настройка для подключения к локальной сети
Открыть страницу Settings/Настройки
Указать 
- Подключение к WiFi сети
- заполнить SSID и PASSWORD
Перезагрузить контроллер
